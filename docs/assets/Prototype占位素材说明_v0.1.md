# Prototype 占位素材说明 v0.1

> 状态：历史归档 / 已废弃，不再作为当前项目素材基线或开发事实。

这份说明只记录 `Prototype 0.1` 早期阶段曾经采用的占位策略：

- 地图背景：`ColorRect`
- 地点建筑：`Area2D`、`ColorRect`、`Label`
