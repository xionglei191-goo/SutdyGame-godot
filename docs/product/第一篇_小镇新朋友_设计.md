# 第一篇：小镇新朋友 设计

## 产品定位

第一篇把孩子从 home 带进 town 的日常冒险：从自己的房间出发，和 Mina 一起看见窗外的小镇，经过医院、机场、铁路站、学校活动房间等地点，慢慢认识新朋友和新地方。

这一篇不做学校应用、课程面板、单词清单或复习测试。孩子看到的是 `Quest Diary` 里的小帮忙、旅行贴纸、声音和颜色发现；英语目标藏在地点、道具和对话里。

## 开场房间探索

`home_room_explore_a` 是可重复的房间灯光探索，第一次完成给 `Room Light Spark` 和 `+1 coin`，之后可重复玩但不重复发放 coin。核心词为 `room / lamp / light`，核心句为 `This is my lamp.` 与 `The light is warm.`。

`home_room_explore_b` 是可重复的窗边小镇观察，第一次完成给 `Window Town Spark` 和 `+1 coin`，之后可重复玩但不重复发放 coin。核心词为 `window / town / look`，核心句为 `Look out the window.` 与 `I can see the town.`。

房间新增可点目标为 `home_lamp`、`home_clock`、`home_window`，继续使用 `data/maps/scene_click_targets_v001.json` 管理，不回到脚本硬编码。

## 小镇新朋友

`town_hospital_bandage_helper`：在 hospital 帮 Mina 递一条 clean bandage。覆盖 greeting 语言：`Hello, can I help?`，并使用通用 `bandage`，不使用任何商标名。

`town_airport_weather_check`：在 airport 给旅行板贴 weather sticker。覆盖 weather 语言：`How is the weather?`、`It is sunny today.`，保持轻量 travel/weather sticker 动作，不改变 park 或 garden routing。

`town_railway_time_stop`：在 railway station 查看 train time。覆盖 time 语言：`What time is it?`，并和现有 `Choose Train Stop` route loop 兼容，time helper 动作在 train stop 已标记后出现。

## 学校活动房间

`music_room` 与 `art_room` 在 prologue 后作为 world_overview PlaceCard 开放，不新增真实子场景。

`school_music_room_sound_find`：孩子在 music room 找到 bright sound，词汇为 `music / sound / listen`。

`school_art_room_color_pick`：孩子在 art room 选择 happy color，词汇为 `art / color / paint`。

这两个地点仍属于 school footprint 内部，是生活冒险地图中的活动房间，不把产品框成学校导览。

## 数据约定

PlaceCard action 使用 `start_dialogue_id` 触发 Mina/NPC 入口对话，dialogue 的 `starts_quest` 指向对应 quest；同时保留 `start_quest_id` 作为兼容字段。按钮可见性使用 `quest_not_completed:<quest_id>` 或既有 starter economy visibility key。

新内容不新增 `data/lessons` fallback，不改变 A-Z memory anchor route order。
